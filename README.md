# Agentic Workforce

Prototype web app for an AI-powered project kickoff room where a human assembles specialist AI agents and runs a structured planning call.

## Run locally

1. Install dependencies with `npm install`
2. Start the app with `npm run dev`
3. Open `http://localhost:3000`

## Prototype flow

- Project dashboard with search and project creation
- Project workspace with notes, recent calls, and draft requirements
- Agent selection modal with role-based AI participants
- Simulated live multi-agent call with hand-raise queue and human override
- Mocked end-call summary with requirements, risks, architecture, UX notes, and next steps
